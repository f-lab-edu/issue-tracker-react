## 소개

F-Lab 프론트엔드 과정 중 세 번째 [**리액트와 타입스크립트로 구현하는 프로젝트**](https://docs.google.com/presentation/d/1bPk_xJgamZvGJuIy9Pg5_4AVKkuBHVMN_xjMdGGLR3g/edit#slide=id.g155725aff19_0_4)입니다.
